3. Website Optimization Suggestions (Task 3)

The following improvements can enhance website performance and user experience:

1. Image Optimization – Compress images to reduce file size and improve loading speed.
2. Lazy Loading – Load images only when they appear on the screen.
3. Minify CSS and JavaScript – Reduce file size to improve performance.
4. Mobile Optimization – Ensure smooth user experience on all devices.
5. Browser Caching – Store website resources locally for faster loading.
